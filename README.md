# README

contoh konfigurasi terraform untuk membuat kubernetes cluster dari awal di lingkungan AWS

konfigurasi ini digunakan untuk workshop bedrock

## Requerement
- aws config
- aws user api key with permission:
   - logs:*
   - iam:*
   - ec2:*
   - eks:*
   - kms:*
